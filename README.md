# Gentech-Java

## About
Gentech is a mobile money application built for Guinea, West Africa. This is the Java/Spring Boot implementation of the same backend I originally built with Python/FastAPI.

## Tech Stack
- Java + Spring Boot
- PostgreSQL
- Redis
- AWS S3
- JWT
- Resend (email notifications)
- Bucket4j (rate limiting)
- Flyway

## Features
- User signup and login
- JWT authentication with refresh tokens
- Two-Factor Authentication (2FA)
- KYC document upload to AWS S3
- Deposit, Withdraw, Transfer
- Rate limiting and IP banning
- Redis caching
- Background jobs
- Pagination
- Full-text search
- Flyway database migrations
- Fraud detection with daily limits
